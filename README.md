# 💕 For KITTU<3 – A Little Universe Made With Love

This is a small romantic website made specially for **KITTU<3** 🌸  
It includes music, floating memories, shooting stars to make wishes, and live timers that count our moments together.

This project is meant to be opened as a **website link**, not just a file.

---

## ✨ Features

- 🔐 Password-protected entry  
- 🎵 Background song (plays from 10s to 60s, loops softly)  
- ⏳ “We’ve been together” live timer  
- 📅 Countdown to the next 17th  
- 💖 Floating hearts  
- 🖼️ Floating photo memories  
- 🌠 Shooting stars to make wishes  
- 💌 Love letters written from the heart  
- 📱 Works on mobile & PC browsers
